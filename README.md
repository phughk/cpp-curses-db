# Cpp Curses DB

A database with a curses interface. Deal with it.

# Install

## Dependencies

1. CMake
1. Vcpkg

## Setup

```
cmake .
make all
```

## Testing

```
make gtest # Not necessary if you ran 'make all'
make test
```

Output of tests is then stored in Testing/Temporary/LastTest.log

## Run

```
./cpp-curses-db
```
